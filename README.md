# Spring Core Basic

Spring Core principles study repository based on lecture examples.

## 학습 내용

- Pure Java 기반 회원/주문/할인 도메인 구현
- 객체 지향 설계 원리 적용
- AppConfig를 통한 관심사 분리
- IoC, DI, Container 개념 학습
- Spring Container와 Spring Bean
- Singleton Container
- Component Scan
- Dependency Injection
- Bean Lifecycle Callback
- Bean Scope

## 핵심 정리

이 프로젝트는 스프링을 사용하기 전에 순수 자바 코드로 객체지향 설계의 문제점을 확인하고,
AppConfig와 Spring Container를 통해 DI, OCP, DIP가 어떻게 적용되는지 학습하는 것을 목표로 한다.
